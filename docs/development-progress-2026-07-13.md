6. 组合风险真实化
   - 自定义仓位权重低于 95% 时，组合风险建议会提示现金缓冲；高于 105% 时提示核对杠杆或重复录入。
   - 组合风险支持输入组合市值，自动估算每个模拟仓位金额和现金缓冲金额。
   - 行业暴露已加入集中度等级、建议上限、超额权重和超额金额，避免现金缓冲场景下误判行业暴露口径。
   - 前端会在本地保存组合市值和模拟仓位输入，刷新页面后自动恢复并重新请求组合风险。
   - 组合风险新增真实持仓输入，支持数量、成本价、市值反推仓位、成本金额、浮盈亏和收益率展示。
   - 前端支持从 CSV/TXT 导入持仓，格式可用 `symbol,shares,cost_price`，导入后自动填充真实持仓输入。
   - 前端支持从 CSV/TXT 导入交易流水，格式可用 `symbol,side,shares,price`，按买入加权成本、卖出平均成本扣减生成当前持仓。
   - 持仓和交易流水导入已支持常见券商中文表头，例如证券代码、持仓数量、成本价、买卖方向、成交数量、成交价格。

## 本地运行配置

后端 `.env` 建议：

```env
STOCK_DOCTOR_DATA_PROVIDER=eastmoney
STOCK_DOCTOR_DATA_REQUEST_TIMEOUT_SECONDS=8
STOCK_DOCTOR_DATA_CACHE_TTL_SECONDS=300
STOCK_DOCTOR_DATA_FRESHNESS_STALE_AFTER_MINUTES=30
STOCK_DOCTOR_TDX_VIPDOC_PATH=E:\new_tdx64\vipdoc
STOCK_DOCTOR_THS_STOCKNAME_PATHS=D:\同花顺软件\同花顺\stockname\stockname_16_0.txt;D:\同花顺软件\同花顺\stockname\stockname_32_0.txt
STOCK_DOCTOR_TUSHARE_TOKEN=<如需 Tushare 财务增强，填入 token>
```

后端启动：

```powershell
cd backend
python -m uvicorn app.main:app --host 127.0.0.1 --port 8010
```

如果已在 `backend` 下创建虚拟环境，也可以用 `.\.venv\Scripts\python -m uvicorn app.main:app --host 127.0.0.1 --port 8010`。

前端启动：

```powershell
cd frontend
npm run dev -- --host 127.0.0.1 --port 30080
```

这些值会通过 `/api/v1/system/data-connectors` 和 `/api/v1/system/runtime-config` 返回，并显示在数据可信度/运行配置面板中。报告导出的 JSON、HTML、Markdown 也会保留这些运行假设，便于离线复盘。
